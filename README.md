# CreatorStore Backend 🚀

A complete E-commerce Backend API built using **Java, Spring Boot, Hibernate, JPA, and PostgreSQL**.

This project provides REST APIs for managing products, orders, and inventory with database integration and clean backend architecture.

---

## ✨ Features

- Product Management APIs
- Order Management APIs
- Stock Quantity Handling
- Input Validation
- PostgreSQL Database Integration
- Layered Architecture
- RESTful APIs
- Swagger API Documentation
- Exception Handling
- JPA & Hibernate ORM

---

## 🛠️ Tech Stack

- Java
- Spring Boot
- Spring Data JPA
- Hibernate
- PostgreSQL
- Maven
- Lombok
- Swagger / OpenAPI
